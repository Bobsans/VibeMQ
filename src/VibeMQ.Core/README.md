# VibeMQ.Core

Core models, enums, interfaces, protocol implementation, and health check server for [VibeMQ](https://github.com/Bobsans/VibeMQ) message broker.

## What's in this package

- Shared types, enums, and interfaces used by VibeMQ.Client and VibeMQ.Server
- Protocol and framing implementation
- Health check HTTP server types

## Installation

```bash
dotnet add package VibeMQ.Core
```

## Documentation

Full documentation: [https://vibemq.readthedocs.io/](https://vibemq.readthedocs.io/)

## Related packages

- **VibeMQ.Server** — run the message broker
- **VibeMQ.Client** — connect and publish/subscribe
