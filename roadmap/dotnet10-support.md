# Поддержка .NET 10

**Описание:** Multi-targeting для поддержки .NET 8 и .NET 10.

**Детальный план:**

## Шаг 1: Обновление целевых фреймворков
**Файл:** `Directory.Build.props`

**Задачи:**
1. Изменить `TargetFramework` на `net8.0;net10.0`
2. Или использовать `TargetFrameworks` для multi-targeting
3. Обновить все `.csproj` файлы

## Шаг 2: Проверка совместимости
**Задачи:**
1. Запустить сборку для обоих фреймворков
2. Исправить ошибки компиляции (если есть)
3. Проверить использование устаревших API

## Шаг 3: Использование новых API .NET 10
**Задачи:**
1. Изучить новые возможности .NET 10
2. Применить оптимизации где возможно:
   - Новые коллекции
   - Улучшения производительности
   - Новые async паттерны

## Шаг 4: Тестирование
**Задачи:**
1. Запустить тесты для обоих фреймворков
2. Проверить совместимость
