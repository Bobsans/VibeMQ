# VibeMQ Roadmap - Детальные планы реализации

Эта папка содержит детальные планы реализации для каждого пункта роадмапа VibeMQ.

## Структура

Каждый пункт роадмапа представлен отдельным файлом:

- [Persistence слой](persistence-layer.md) - Постоянное хранилище сообщений (SQLite, RocksDB, PostgreSQL)
- [Кластеризация](clustering.md) - Горизонтальное масштабирование и отказоустойчивость
- [Поддержка других протоколов](protocol-support.md) - AMQP, MQTT, HTTP REST, WebSocket
- [Расширенный мониторинг](monitoring.md) - Prometheus, Grafana, OpenTelemetry, Audit logging
- [Веб-интерфейс управления](web-ui.md) - Веб-админка для мониторинга и управления
- [Сжатие на уровне протокола](compression.md) - Опциональное сжатие сообщений
- [Гранулярная авторизация](authorization.md) - Детальная система разрешений
- [Бинарный протокол](binary-protocol.md) - Альтернативный бинарный формат

## Формат планов

Каждый файл содержит:

- **Описание** - краткое описание задачи
- **Текущее состояние** - что уже есть в проекте (если применимо)
- **Детальный план** - разбивка на шаги с:
  - Файлами для изменения
  - Конкретными задачами
